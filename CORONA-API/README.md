This project is a COVID-19 tracker for India that provides up-to-date information about the coronavirus situation in all states and union territories. The application fetches real-time data from a COVID-19 API and displays it in a user-friendly interface.

Key Features:

Shows the number of positive cases, discharged/recovered cases, and deaths for each state and union territory.

Displays foreign cases affected in India.

Line graph visualization to show trends and patterns over time.

Built using HTML, CSS, Bootstrap, and JavaScript for a responsive and interactive interface.

Handles cases where data might not be available with proper user notifications.

Technologies Used:

HTML & CSS: For structure and styling of the webpage.

Bootstrap: For responsive design and layout.

JavaScript: For fetching API data, dynamically updating the webpage, and rendering charts.

Chart.js: To create interactive line graphs for COVID-19 trends.

This project helps users quickly understand the COVID-19 scenario in India and track state-wise updates in a clear, visual format.
